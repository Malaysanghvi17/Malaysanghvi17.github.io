Follow the below given steps to upload docker image to dockerhub

  1. Step 1: Create a Dockerhub account if you don't have one already.
  2. Step 2: Create your project, for example, here I am having a simple project which shows the todo list using Node.js.
  3. Step 3: Add Dockerfile to your project.
  4. Step 4: Build image of the project you just created.
  5. Step 5: Tag your Docker image with your Dockerhub username and repository name like: docker tag imagename yourusername/imagename:tag.
  6. Step 6: You can check that new image will be created with username/repositoryname.
  7. Step 7: Log in to Dockerhub using the Docker CLI: docker login.
  8. Step 8: Push your Docker image to Dockerhub: docker push yourusername/repositoryname:tag.
  9. Step 9: Verify that your image is uploaded successfully on Dockerhub.
